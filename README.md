# Cross-Fire
"Cross Fire" is a game write in pygame  
## Game manipulation  
#### How to play?  
the plane:  
●↑：turn up  
●↓: turn down  
●←: turn right  
●→: turn left  
the dragon:  
Press the keyboard B  
If a dragon hits a bullet，Dragon will die.If dragons encounter enemies，The enemy will die.
